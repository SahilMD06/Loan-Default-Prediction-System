# Loan-Default-Prediction-System


A complete end-to-end Machine Learning project that predicts whether a loan applicant is likely to repay a loan or default based on historical loan application data. The project follows a professional data science workflow, including data preprocessing, exploratory data analysis, feature engineering, model training, evaluation, hyperparameter tuning, and model deployment preparation.

#### Project Objectives

* Analyze loan applicant data to identify factors influencing loan repayment.
* Build and compare multiple classification models for loan default prediction.
* Improve prediction performance through feature engineering and hyperparameter tuning.
* Provide business insights to support loan approval decisions and risk management.

#### Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Joblib
* Pickle

#### Project Workflow

1. Data Loading and Inspection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Building

   * Logistic Regression
   * Decision Tree Classifier
   * Random Forest Classifier
6. Model Evaluation

   * Accuracy
   * Precision
   * Recall
   * F1 Score
   * Confusion Matrix
   * Classification Report
7. Model Comparison
8. Feature Importance Analysis
9. Hyperparameter Tuning using GridSearchCV
10. ROC Curve and AUC Analysis
11. Cross Validation
12. Learning Curve Analysis
13. Model Saving and Loading
14. Sample Predictions and Business Recommendations

#### Key Features

* Missing value handling and outlier treatment
* Label Encoding and One-Hot Encoding
* Feature Scaling using StandardScaler
* Advanced visualizations with business insights
* Feature engineering using Total Income and EMI estimation
* Random Forest feature importance analysis
* Hyperparameter optimization for improved performance
* Model persistence using Joblib and Pickle
* Ready for internship submission, GitHub portfolio, and academic projects

#### Business Impact

The system helps financial institutions identify high-risk loan applicants, reduce potential defaults, improve loan approval decisions, and support data-driven risk assessment strategies.

#### Dataset

The project uses a loan application dataset containing applicant demographics, income details, loan information, credit history, and loan approval status.

#### Outcome

A robust machine learning solution capable of predicting loan repayment likelihood and assisting lenders in making informed lending decisions.
